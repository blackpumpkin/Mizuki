# 阅读

![](http://pic.962.net/up/2024-5/20245611038216.jpg)

阅读 APP 是一款开源的阅读软件，如今的小说阅读软件总是在不断的添加广告,但这款APP不用再担心广告,开源的软件代码经过大量修改,更符合使用习惯。阅读只是一个转码工具，不提供内容，第一次安装app，需要自己手动导入书源。
平台：<font color=red>Android</font>
主要功能：
++ 自定义书源，自己设置规则，抓取网页数据，规则简单易懂，软件内有规则说明。
++ 列表书架，网格书架自由切换。
++ 书源规则支持搜索及发现，所有找书看书功能全部自定义，找书更方便。
++ 订阅内容,可以订阅想看的任何内容,看你想看
++ 支持替换净化，去除广告替换内容很方便。
++ 支持本地TXT、EPUB阅读，手动浏览，智能扫描。
++ 支持高度自定义阅读界面，切换字体、颜色、背景、行距、段距、加粗、简繁转换等。
++ 支持多种翻页模式，覆盖、仿真、滑动、滚动等。
++ 软件开源，持续优化，无广告。

相关链接：

使用说明（必看）：<https://www.yuque.com/legado/wiki>
官方网站：<https://www.yiove.com/>
Yiove书源仓库：<https://shuyuan.yiove.com/>
Yiove论坛：<https://bbs.yiove.com/>

# tachiyomi （目前已停止维护）

![](http://img3.pcsoft.com.cn/capmobile/202401/131031-65b33ec7a1fd5.jpg)
平台：<font color=red>Android</font>
tachiyomi是一个开源漫画软件，是需要安装扩展插件才能使用的，就跟导入图源、漫画源差不多。软件本身是看不了漫画的，相当于就是一个空壳；但是，只要安装了不同的中文扩展插件，就可以看对应漫画网站的内容了，相当于是一个app聚合所有漫画平台。**目前由于版权炮已停止维护更新。**

主要功能：
++ 包含多种查看功能和设置的可配置的阅读器
++ 在线和离线阅读
++ 从下一个未读章节继续阅读
++ 支持MyAnimeList
++ 定时搜索更新
++ 过滤章节
++ 通过分类来管理漫画库
++ 支持自动存储阅读记录
++ 支持漫画收藏功能
++ 支持漫画关键词搜索功能

# mihon

![](http://pic.2265.com/upload/2024-1/20241311529216981.png)
平台：<font color=red>Android</font>
Mihon 是一款安卓端的免费、开源漫画阅读器，支持漫画在线阅读与下载。Mihon不生产漫画，而是通过安装插件，让你能获取到一个个图源站里的内容。应用界面清爽干净，操作简单，纯净无广告。操作界面和使用逻辑不能说是跟 Tachiyomi相似，简直是一模一样。

添加图源方法：
**方法一：**
打开设置 → 浏览 → 插件仓库 → 添加，将`https://raw.githubusercontent.com/keiyoushi/extensions/repo/index.min.json`添加至插件仓库中。（需要科学环境）
返回主界面 - 浏览 - 插件，下拉刷新，就能看到图源插件了。
可以点击右上角语言按钮进行筛选，只勾选显示中文、中文简体。
选择需要的插件安装，点击信任后，就能愉快地看漫画了！

**方法二：**
手动下载和安装图源扩展插件。
这些图源插件，其实就是一些 apk 安装包，我们可以手动下载安装。
插件下载网站：<https://keiyoushi.github.io/extensions/>
筛选想要使用的插件，单独下载到手机上安装。
也可以在 GitHub 打包下载全部插件，再选择安装。下载压缩包后解压，插件都在 apk 文件夹里。
GitHub 仓库地址：<https://github.com/keiyoushi/extensions/>
