### 本地文件类

#### Chinese (Simplified) (简体中文) Language Pack for Visual Studio

顾名思义，简体中文的汉化包。

#### Code Runner

VS Code上的代码运行插件，安装后右上角会出现三角箭头, 点击即可运行当前代码，或者使用快捷键`ctrl + alt + N`；停止当前代码：`Ctrl + Alt  + M`

#### Code Translate

划词翻译，鼠标划词悬浮在要查的单词上，很快就能出现翻译。

#### Fix VSCode Checksums

有时安装其它插件会显示**code损坏**，安装此插件即可解决。

#### vscode-icons

官方出品的对各种类型的文件图标进行优化显示，这样就可以直接看文件的图标，从而快速知道文件的类型，而不是去看文件的后缀。

#### Material Icon Theme

另一款文件图标优化插件，比官方出品要多一些。

### 效率提升类

#### Bookmarks

安装后，将光标置于想要添加书签的位置，右键——书签——开关，即可标记本行，在左侧点击点击Bookmarks图标，便可以快速查阅标记的书签。

#### GitLens

安装后，每行代码的末尾，都会公开最后一次提交的作者、时间、信息。鼠标悬停在这些注释上显示更多详细信息。

#### Import Cost

可以显示在VS代码编辑器中导入的程序包的大小。

#### indent-rainbow

可以把缩进的空格填满颜色高亮处理，一眼看出每一级的缩进。

#### Prettier - Code formatter

自动格式化代码。

#### Colorize

安装后，所有颜色代码都将显示其背景色。

#### Image preview

通过此插件，当鼠标悬浮在图片的链接上时，可以实时预览该图片，并看到图片的大小和分辨率。

### Markdown类

#### Markdown All in One

用于提供Markdown编辑的全方位支持，包括实时预览、语法提示、目录生成、表格生成等多种功能。

#### Markdown Preview Enhanced

安装后，点击右上角的**MPE:打开侧边预览**，可显示markdown被渲染的页面，我们便可以一边编辑内容，一边查看文件内容的效果。
另外，具备导出 PDF、支持数学公式、流程图等多种高级功能。

#### Markdown Preview Github Styling

改善和优化Markdown预览功能。

#### markdownlint

检查markdown的语法错误。

### HTML、CSS、JS类

#### Auto Rename Tag

自动重命名成对的 HTML/XML 标签。

#### Live Server

通过内置的本地服务器，可以实时显示 HTML、CSS 和 Java 文件的修改，简化了网页开发和测试过程。

### 其它文件类型

#### Ini for VSCode

ini文件扩展插件，可以对ini代码进行折叠。

#### vscode-pdf

可以在vscode中直接浏览pdf文件。

### 摸鱼/杂项类

#### z-reader

支持在线阅读和本地阅读。

#### 韭菜盒子

vscode中查看股价涨跌。

#### Cloudmusic

简易的网易云音乐，包括账户登录、歌单、查看评论、歌词等功能。
